# Development

this dir contains the development files for the project.

## Workflow

1. user create tasks in `docs/development/YY-MM-DD/user.md`
2. ask agent to create a `outline-plan.md` and review the plan
3. ask agent to create a `detailed-plan.md` based on the outline plan and review the plan
4. ask agent to complete the task based on the detailed plan, and review the result
